# 🐍 Serpent Surge X – Java OOP Snake Game Project

Serpent Surge X is a feature-rich Snake Game developed as a final project for the 2nd Semester Object-Oriented Programming (OOP) course using Java. The project demonstrates the practical application of object-oriented design principles, event-driven programming, graphical user interface development, and game logic implementation.

The game is inspired by the classic Snake arcade game, where players control a snake that moves continuously across the game board. The objective is to collect food items to increase the snake's length and score while avoiding collisions with walls, obstacles, and the snake's own body. As the game progresses, the challenge increases, requiring faster reflexes and strategic movement.

The project was designed using Java's OOP concepts, including classes, objects, encapsulation, inheritance, and modular programming. The game architecture separates different responsibilities such as game management, player controls, food generation, scoring, collision detection, and graphical rendering, making the code more organized, maintainable, and scalable.
